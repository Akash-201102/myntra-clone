# myntra-clone
A clean and modern Myntra-like static shopping UI built using only HTML and CSS. It includes a sticky header, search bar, hero section, filters sidebar, and responsive product grid. The layout adapts for mobiles and tablets, showcasing trendy products with ratings, pricing, and a minimal design.

# Myntra

This is a clean and modern Myntra-inspired shopping website built using only HTML and CSS. The project includes a hero banner, filters sidebar, responsive product grid, and a simple e-commerce layout. It is designed purely for frontend practice and to help understand layout structure, styling, and responsive design without using frameworks.

## Features
- Stylish Myntra-like layout  
- Sticky header with brand and actions  
- Hero section with image and CTA buttons  
- Filters for brand and price (UI only)  
- Responsive product grid (4 → 3 → 2 → 1 columns)  
- Product cards with images, ratings, and price details  
- Pure HTML + CSS, no JavaScript needed

## How to Run
1. Download or clone the repository  
2. Open the `index.html` file in any browser  
3. That’s it — the UI loads instantly

## Technologies Used
- HTML5  
- CSS3 (Flexbox + Grid)

## Purpose
This project is created for learning and frontend practice — suitable for portfolios, GitHub uploads, and UI design improvement.
